Lời Giải Chi Tiết Các Bài Toán Không Gian Oxyz

Câu 1

Đề bài: Trong không gian tọa độ Oxyz, cho mặt phẳng (P): 2x + 6y - 3z + 4 = 0. Mặt phẳng (P) đi qua điểm nào trong các điểm sau:

  • A. (-1; 1; 4)
  • B. (1; 0; 2)
  • C. (1; -1; 4)
  • D. (1; 1; 0)

Cách làm:

Để kiểm tra, ta thay tọa độ từng điểm vào phương trình mặt phẳng:

  • Điểm A (-1; 1; 4): 2(-1) + 6(1) - 3(4) + 4 = -2 + 6 - 12 + 4 = -4 ≠ 0. Không thuộc.
  • Điểm B (1; 0; 2): 2(1) + 6(0) - 3(2) + 4 = 2 + 0 - 6 + 4 = 0. Thuộc.
  • Điểm C (1; -1; 4): 2(1) + 6(-1) - 3(4) + 4 = 2 - 6 - 12 + 4 = -12 ≠ 0. Không thuộc.
  • Điểm D (1; 1; 0): 2(1) + 6(1) - 3(0) + 4 = 2 + 6 + 0 + 4 = 12 ≠ 0. Không thuộc.

Đáp án: B

Câu 2

Đề bài: Trong không gian Oxyz, phương trình nào là phương trình mặt phẳng qua M(1; 2; -3) và có vectơ pháp tuyến n⃗ = (1; -2; 3)?

  • A. x - 2y + 3z + 12 = 0
  • B. x - 2y - 3z + 6 = 0
  • C. x - 2y + 3z - 12 = 0
  • D. x - 2y - 3z - 6 = 0

Cách làm:

Phương trình mặt phẳng: a(x - x₀) + b(y - y₀) + c(z - z₀) = 0. Với M(1; 2; -3) và n⃗ = (1; -2; 3):

1(x - 1) + (-2)(y - 2) + 3(z - (-3)) = x - 1 - 2y + 4 + 3z + 9 = x - 2y + 3z + 12 = 0.

Đáp án: A

Câu 3

Đề bài: Trong không gian Oxyz, cho M(1; 2; 3). Mặt phẳng (P): x + Ay + Bz + C = 0 chứa trục Oz và cách M khoảng lớn nhất. Tính A + B + C.

Cách làm:

Chứa trục Oz ⇒ C = 0. Phương trình: x + Ay + Bz = 0.

Khoảng cách: d = |1 + 2A + 3B| / √(1 + A² + B²). Tối đa khi (1; A; B) ∥ (1; 2; 3) ⇒ A = 2, B = 3.

x + 2y + 3z = 0, A + B + C = 2 + 3 + 0 = 5.

Đáp án: 5

Câu 4

Đề bài: Trong không gian Oxyz, cho G(2; -2; 1). Mặt phẳng (P) qua G, cắt Ox, Oy, Oz tại A, B, C sao cho G là trọng tâm △ABC. Phương trình là:

  • A. x/6 + y/(-6) + z/3 = 0
  • B. x/2 + y/(-2) + z/1 = 1
  • C. x - y + 2z - 6 = 0
  • D. 2x - 2y + z - 6 = 0

Cách làm:

A(a; 0; 0), B(0; b; 0), C(0; 0; c). Trọng tâm: G = (a/3; b/3; c/3) = (2; -2; 1) ⇒ a = 6, b = -6, c = 3.

Phương trình: x/6 + y/(-6) + z/3 = 1 ⇔ x - y + 2z - 6 = 0.

Đáp án: C

Câu 5

Đề bài: Mặt phẳng (P) qua M(1; -2; 3), song song Oy, vuông góc (Q): 3x - y + 4z + 1 = 0. Kiểm tra:

  1. Vectơ pháp tuyến n⃗ = (4; 0; -3). Đúng/Sai
  2. (P) qua N(0; 1; -1). Đúng/Sai
  3. (P) không qua A(2; 0; -2). Đúng/Sai
  4. Khoảng cách từ gốc đến (P) bằng 1. Đúng/Sai

Cách làm:

Song song Oy ⇒ n⃗ = (a; 0; c). Vuông góc (Q) (n⃗_Q = (3; -1; 4)) ⇒ 3a + 4c = 0. Chọn a = 4, c = -3.

Phương trình qua M: 4x - 3z + 5 = 0.

  1. n⃗ = (4; 0; -3). Đúng
  2. Thay N: 4(0) - 3(-1) + 5 = 8 ≠ 0. Sai
  3. Thay A: 4(2) - 3(-2) + 5 = 19 ≠ 0. Đúng
  4. Khoảng cách: |5|/√(16 + 9) = 1. Đúng

Câu 6

Đề bài: Cho A(-1; 0; 2), (P): 2x + y - 2z = 0. Kiểm tra:

  1. n⃗ = (2; 1; -2) và qua A. Đúng/Sai
  2. Mặt phẳng qua A, song song (P): 2x + y - 2z + 6 = 0. Đúng/Sai
  3. Mặt phẳng qua O, A, vuông góc (P): 2x - 2y + z = 0. Đúng/Sai
  4. Có 2 mặt phẳng song song (P), cách A bằng 2. Đúng/Sai

Cách làm:

  1. n⃗ đúng. Thay A: -6 ≠ 0. Sai
  2. Qua A: 2x + y - 2z + 6 = 0. Đúng
  3. n⃗_P × OA = (2; -2; -1). Phương trình khớp. Đúng
  4. Khoảng cách từ A đến (P): 2. Có 2 mặt phẳng. Đúng

Câu 7

Câu 10

Câu 16

Đề bài: Mặt phẳng (P) qua A(1; -2; -2), vuông góc (Oxz), khoảng cách từ B(3; 1; -3) đến (P) là 3/5. Dạng 2x + by + cz + d = 0. Tính P = 3b - 2c - d.

Cách làm:

Vuông góc (Oxz) ⇒ n⃗ = (2; 0; c). Qua A: 2x + 2z + 2 = 0. Khoảng cách từ B khớp. b = 0, c = 2, d = 2. P = -6.

Đáp án: -6

Câu 17

  1. Mọi x + 2y - z + m = 0 song song (α). Đúng/Sai
  2. (P) vuông góc (Q): 2x - 3y - 4z = 0. Đúng/Sai
  3. ad = 15. Đúng/Sai
  4. Qua M(1; 2; -1). Đúng/Sai

Cách làm:

(P): x + 2y - z + 5 = 0.

  1. Đúng
  2. n⃗_P · n⃗_Q = 0. Đúng
  3. ad = 5 ≠ 15. Sai
  4. Thay M: 11 ≠ 0. Sai

Câu 20

  1. (P) và (Q) vuông góc. Đúng/Sai
  2. M(0; -3; 0) cách đều (P), (Q). Đúng/Sai
  3. Mặt phẳng vuông góc cả hai có n⃗ = (a; b; -2), a + b = -2. Đúng/Sai
  4. (R): x + y - z + 2 = 0 song song (P), cách 3. Đúng/Sai

Cách làm:

  1. n⃗_P · n⃗_Q ≠ 0. Sai
  2. Khoảng cách không bằng nhau. Sai
  3. n⃗ = (0; -2; -2), a + b = -2. Đúng
  4. Khoảng cách = 1/√3 ≠ 3. Sai
